ViPlex Express

O ViPlex é uma solução indicada pela Intelbras e projetada para o gerenciamento de conteúdo em displays LED, permitindo a edição e reprodução de informações de forma eficiente em ambientes corporativos e comerciais. Este software é compatível com as controladoras da linha TB, incluindo modelos como a TB20 PLUS, TB2, TB50 e TB60.

Com uma plataforma web interativa, o ViPlex oferece aos operadores a capacidade de personalizar a ambientação conforme as necessidades específicas de cada perfil, garantindo uma experiência de uso intuitiva e confortável. A interface é amigável e facilita o acesso a diversas funcionalidades.

Para acessar a Interface Web do ViPlex, é necessário que o dispositivo esteja conectado à mesma rede que o computador. O acesso é feito através do endereço IP do dispositivo, simplificando a gestão e a atualização de conteúdos em tempo real.

NovaLCT

O NovaLCT é uma solução da Novastar projetada para o gerenciamento de conteúdo em displays de LED, calibração e ajustes avançados. Este software é compatível com as controladoras da linha TB, incluindo modelos como a TB20 PLUS, TB2, TB50 e TB60.

HARDWARE

Via Cabo de Rede

Essa configuração utiliza uma conexão física entre o painel LED e o computador para garantir estabilidade e alta velocidade de transmissão. Siga os passos abaixo:

  1. Conecte o Cabo de Rede:

    Insira uma extremidade do cabo de rede no painel LED e a outra no roteador ou diretamente no computador.

  2. Configuração de Rede:

    No computador, certifique-se de que a conexão está configurada para obter um endereço IP automaticamente (DHCP).

  3. Abra o Software:

    Inicie o ViPlex Express ou NovaLCT e certifique-se de que o dispositivo foi detectado na lista de terminais.

  4. Teste a Conexão:

    Envie um conteúdo de teste para verificar se a comunicação entre o computador e o painel está funcionando corretamente.

LAN com Fio

A configuração LAN com fio é ideal para redes locais e ambientes corporativos. Siga os passos abaixo:

  1. Conecte o Cabo de Rede:

    Insira uma extremidade do cabo de rede no painel LED e a outra no switch ou roteador da rede local.

  2. Desative o DHCP:

    No painel LED, desative o serviço DHCP e configure um endereço IP estático.

  3. Configuração de Rede no Computador:

    No computador, configure um endereço IP na mesma faixa do painel LED.

  4. Conexão com o Software:

    Abra o ViPlex Express ou NovaLCT e conecte-se ao painel LED utilizando o endereço IP configurado.

  5. Verificação:

    Envie um conteúdo de teste para garantir que o painel LED está respondendo corretamente.

Wi-Fi AP

O modo Wi-Fi AP permite que o painel LED funcione como um ponto de acesso, facilitando conexões sem fio. Siga os passos abaixo:

  1. Localize o SSID:

    Encontre o SSID do painel LED (geralmente "AP" + últimos 8 dígitos do número de série). Essa informação está disponível na etiqueta do dispositivo.

  2. Conecte-se ao Wi-Fi:

    No computador ou dispositivo móvel, conecte-se ao Wi-Fi utilizando o SSID e a senha padrão do painel LED.

  3. Abra o Software:

    Inicie o ViPlex Express ou o aplicativo móvel ViPlex Handy e localize o painel LED na lista de dispositivos.

  4. Configuração Inicial:

    Acesse as configurações do painel e, se necessário, altere o SSID ou a senha para maior segurança.

  5. Teste a Conexão:

    Envie um conteúdo simples para o painel LED e confirme se a transmissão está funcionando.

Login

ViPlex Express

1. Abra o ViPlex Express e vá para a página Screens.

2. Clique em Refresh para atualizar a lista de dispositivos.

3. Selecione a controladora e clique em Connect. Digite a senha padrão "admin" e clique em OK.

NovaLCT

1. Abra o NovaLCT e vá para o menu User > Media Player Login.

2. Na caixa de diálogo Terminal List, clique em Refresh.

3. Selecione a controladora e clique em Connect System. Digite o nome de usuário "admin" e a senha padrão.

ViPlex Handy

1. Abra o ViPlex Handy e vá para a página Local Devices.

2. Atualize a lista de dispositivos locais e selecione a controladora desejada. Clique em Connect e insira as credenciais.

Tela Inicial ViPlex Express

Após a instalação e login bem-sucedido, a página inicial do ViPlex Express exibe as controladoras disponíveis, um menu superior de navegação e atalhos rápidos. Essa interface permite o gerenciamento completo das funções do dispositivo de maneira intuitiva.

Menu Superior

O menu superior é o ponto de acesso para as três principais áreas de funcionalidade do ViPlex Express:

  • Screens: Exibe todos os dispositivos conectados, com informações como status de conexão, nome do dispositivo e armazenamento disponível. É o ponto de partida para conectar e monitorar as controladoras.
  • Solutions: A área dedicada à criação, edição e organização de conteúdos para os displays LED, incluindo imagens, vídeos e layouts personalizados.
  • Control: Permite o ajuste das preferências e configurações avançadas do software e da controladora, como idioma, sincronização de horário, controle de brilho e atualizações.

Painel "Screens" (Visão Geral)

Localizado na página inicial, este painel oferece uma visão rápida do status dos seus dispositivos:

  • **Status da conexão:** Indica se as controladoras estão online, offline, não logadas ou o total de dispositivos.
  • **Informações do dispositivo:** Mostra o endereço IP, nome e resolução de cada controladora.
  • **Controladoras em execução:** Lista as controladoras que estão ativas e reproduzindo conteúdo.

Área "Solutions" (Criação e Gestão de Conteúdo)

A área "Solutions" é onde você gerencia os programas que serão exibidos nos painéis de LED. É aqui que novos layouts são criados e o conteúdo é organizado.

Adicionar layout de exibição (New -> Regular Screen)

Esta função permite criar um novo programa ou "solução" para o seu painel de LED. Você pode combinar diferentes tipos de mídia (imagens, vídeos, textos) em um layout personalizado e definir como eles serão exibidos.

Passo a passo para criar um novo layout:
  1. Acessar "Solutions": No menu superior, clique em **"Solutions"**.
  2. Criar Novo Programa: Clique no botão **"New"** e selecione **"Regular Screen"** no menu suspenso.
  3. Definir Propriedades do Layout:
    • Insira um **Nome** para o seu layout.
    • Defina a **Resolução** (largura x altura) do layout, que deve corresponder à resolução do seu painel de LED para melhor qualidade.
    • Clique em `OK`.
  4. Adicionar Elementos de Mídia:
    • No editor de layout que se abre, você verá uma tela em branco. À direita ou em um painel lateral, haverá opções para adicionar mídia.
    • Imagens e Vídeos: Clique no ícone de "Imagem" ou "Vídeo" e navegue até os arquivos em seu computador. Arraste e solte-os na tela do layout.
    • Texto: Clique no ícone de "Texto" para criar uma caixa de texto. Digite sua mensagem, ajuste fonte, cor, tamanho e posição.
    • Outros Widgets: Adicione elementos como relógios digitais, contadores, ou transmissões de RSS, se disponíveis.
  5. Organizar e Editar Elementos:
    • Clique e arraste os elementos para ajustar sua posição e tamanho no layout.
    • Para cada elemento, você pode ajustar propriedades como duração de exibição, efeitos de entrada/saída, e camadas (para sobrepor elementos).
  6. Salvar o Layout: Após configurar, clique em **"Save"** para guardar suas alterações.

Área "Control" (Configurações e Monitoramento)

A área "Control" oferece acesso a diversas configurações avançadas e ferramentas de monitoramento para o software e as controladoras de LED.

Play logs

O recurso "Play logs" permite visualizar um relatório detalhado sobre a reprodução das mídias em suas controladoras. É uma ferramenta essencial para auditoria e controle de conteúdo.

Como utilizar Play logs:
  1. Acessar Play logs: No menu superior, clique em **"Control"** e selecione **"Play logs"**.
  2. Consultar Logs: Na tela de logs, você pode filtrar por data, hora ou controladora para encontrar informações específicas.
  3. Visualizar Detalhes: Cada entrada de log mostra qual mídia foi exibida, em qual data e horário, e em qual controladora.
  4. Exportar Relatório: Clique na opção **"Export"** para salvar o relatório em formato CSV ou Excel, facilitando a análise externa e o arquivamento.

Playback Management

Gerencie o status de reprodução e assegure que o conteúdo mais recente seja exibido. É onde você verifica se as mídias foram enviadas e estão ativas nos displays.

Como utilizar Playback Management:
  1. Acessar Playback Management: No menu superior, clique em **"Control"** e selecione **"Playback Management"**.
  2. Atualizar Conteúdo: Selecione a controladora e clique em **"Update Content"** para enviar as últimas alterações de layout ou mídia.
  3. Verificar Status: A tela mostra o status de cada programa (em reprodução, pausado, erro) e o progresso da transferência.
  4. Visualizar Screenshot: Use a opção **"View Screenshot"** para obter uma captura de tela em tempo real do que está sendo exibido no painel de LED, confirmando que o conteúdo está correto.

Network Configuration

Configure as opções de rede das controladoras, seja para conectividade com ou sem fio. Essencial para garantir a comunicação entre o software e o painel de LED.

Como utilizar Network Configuration:
  1. Acessar Network Configuration: No menu superior, clique em **"Control"** e selecione **"Network Configuration"**.
  2. Configurar Ethernet (Cabo de Rede):
    • Para conexão via cabo, você pode definir se a controladora obterá um endereço IP automaticamente (DHCP) ou configurar um IP estático.
    • Um IP estático garante que o endereço da controladora não mude, o que é ideal para acesso remoto.
  3. Configurar Wi-Fi:
    • Se a controladora suporta Wi-Fi, você pode configurá-la como Ponto de Acesso (AP) ou conectá-la a uma rede Wi-Fi existente (Modo Cliente/STA).
    • No modo AP, a controladora cria sua própria rede Wi-Fi para conexão direta.
    • No modo Cliente/STA, a controladora se conecta à rede Wi-Fi local existente.
  4. Verificar Conexão: Após as alterações, verifique o status da conexão para garantir que a controladora está online.

Brightness Adjustment

Ajuste o brilho do painel de LED para adequar-se às condições de luminosidade do ambiente ou às suas preferências. Isso pode ser feito manualmente ou de forma programada.

Como utilizar Brightness Adjustment:
  1. Acessar Brightness Adjustment: No menu superior, clique em **"Control"** e selecione **"Brightness Adjustment"**.
  2. Ajuste Manual:
    • Use uma barra deslizante para aumentar ou diminuir o brilho em tempo real.
    • Ideal para ajustes rápidos e pontuais.
  3. Ajuste Programado (Automático):
    • Configure horários específicos para que o brilho seja ajustado automaticamente ao longo do dia (ex: mais intenso durante o dia, mais suave à noite).
    • Se houver um sensor de luminosidade conectado, configure o ajuste automático baseado na luz ambiente para uma experiência dinâmica e economizar energia.

Video Source

Controle a fonte de entrada de vídeo da controladora, definindo qual conteúdo será exibido no painel de LED e em qual resolução.

Como utilizar Video Source:
  1. Acessar Video Source: No menu superior, clique em **"Control"** e selecione **"Video Source"**.
  2. Selecionar Fonte de Entrada:
    • Escolha entre as fontes de vídeo disponíveis, como a mídia interna da controladora ou uma entrada externa (ex: HDMI, se disponível e conectado).
  3. Ajustar Resolução:
    • Configure a resolução e a taxa de atualização para a fonte de vídeo selecionada, garantindo que correspondam às especificações do seu painel e do conteúdo.

Screen Status Control

Gerencie o status de exibição do seu painel de LED, permitindo desligá-lo temporariamente (blackout) ou programar seu funcionamento em horários específicos.

Como utilizar Screen Status Control:
  1. Acessar Screen Status Control: No menu superior, clique em **"Control"** e selecione **"Screen Status Control"**.
  2. Controle Manual:
    • Ative a opção "Blackout" para desligar a exibição da tela imediatamente, ou desative para retornar à exibição normal.
    • Útil para pausas rápidas ou interrupções.
  3. Controle Programado:
    • Programe horários de início e fim para o modo "Blackout", ou para ligar/desligar o painel, otimizando o uso de energia e a vida útil do display.
    • Por exemplo, defina para o painel entrar em blackout fora do horário comercial.

Restart Configuration

Configure reinícios automáticos para a controladora em intervalos regulares ou horários específicos, o que pode ajudar a manter o desempenho do sistema e resolver pequenos problemas.

Como utilizar Restart Configuration:
  1. Acessar Restart Configuration: No menu superior, clique em **"Control"** e selecione **"Restart Configuration"**.
  2. Reinício Imediato: Opção para reiniciar a controladora instantaneamente.
  3. Reinício Programado:
    • Defina uma programação diária ou semanal para a controladora reiniciar automaticamente.
    • Um reinício regular pode liberar memória e otimizar o funcionamento do sistema, evitando falhas a longo prazo.

Color Temperature

A controladora oferece opções de temperatura de cor para ajustar o balanço de branco do painel de LED, impactando a tonalidade geral da imagem.

Como utilizar Color Temperature:
  1. Acessar Color Temperature: No menu superior, clique em **"Control"** e selecione **"Color Temperature"**.
  2. Escolher uma Opção:
    • **Natural White:** Um balanço de cores mais neutro e próximo à luz do dia.
    • **Standard White:** Uma configuração padrão, geralmente ligeiramente mais fria que o natural.
    • **Cold White:** Uma tonalidade mais azulada, que pode ser útil em ambientes externos com forte luz solar.
  3. Aplicar: Selecione a opção desejada para alterar a temperatura de cor do painel.

Monitoring

Monitore o desempenho e o status de hardware da controladora, incluindo o uso de recursos e informações de sensores, se houver.

Como utilizar Monitoring:
  1. Acessar Monitoring: No menu superior, clique em **"Control"** e selecione **"Monitoring"**.
  2. Verificar Uso de Recursos:
    • Visualize gráficos ou indicadores de uso de memória (RAM) e processador (CPU) da controladora.
    • Um alto uso pode indicar problemas de desempenho ou sobrecarga de mídia.
  3. Gerenciar Espaço de Armazenamento:
    • Monitore o espaço de armazenamento interno da controladora e use a opção para limpar arquivos de mídia desnecessários, liberando espaço.
  4. Sensores (se conectado):
    • Se um sensor de luminosidade ou temperatura estiver conectado, esta seção exibirá as leituras em tempo real, permitindo ajustes dinâmicos com base no ambiente.

Font Management

Gerencie as fontes de texto disponíveis para uso em seus layouts, permitindo adicionar novas fontes instaladas no computador.

Como utilizar Font Management:
  1. Acessar Font Management: No menu superior, clique em **"Control"** e selecione **"Font Management"**.
  2. Visualizar Fontes Existentes: A tela exibirá as fontes que já estão instaladas na sua máquina e disponíveis para o software.
  3. Adicionar Novas Fontes:
    • Para usar uma fonte específica que não está na lista, certifique-se de que ela esteja instalada no sistema operacional do seu computador.
    • O ViPlex Express pode então detectá-la e permitir seu uso nos layouts de texto.

Player Upgrade

Verifique e instale atualizações de firmware para a controladora de LED, garantindo acesso às últimas funcionalidades, melhorias de desempenho e correções de segurança.

Como utilizar Player Upgrade:
  1. Acessar Player Upgrade: No menu superior, clique em **"Control"** e selecione **"Player Upgrade"**.
  2. Verificar Atualizações Online: O software pode procurar por atualizações disponíveis diretamente dos servidores da Novastar.
  3. Atualizar Localmente:
    • Se você já possui um arquivo de firmware baixado (geralmente do site da Novastar), pode realizar a atualização offline.
    • Siga as instruções na tela para selecionar o arquivo e iniciar o processo de upgrade.
  4. Importante: Não desligue a controladora durante o processo de atualização.

Player Information

Visualize detalhes importantes sobre o hardware e software da controladora de LED, útil para suporte técnico e identificação do dispositivo.

Como utilizar Player Information:
  1. Acessar Player Information: No menu superior, clique em **"Control"** e selecione **"Player Information"**.
  2. Verificar Dados: A tela exibirá informações como:
    • Número de Série (S/N) da controladora.
    • Versão do firmware (software) instalado.
    • Modelo de hardware.
    • Outros dados técnicos relevantes.

Multi-Screen Mosaic

Para controladoras que suportam múltiplas saídas HDMI (como TB30, TB50, TB60), este recurso permite criar um mosaico de telas, combinando vários painéis de LED para formar uma única tela maior.

Como utilizar Multi-Screen Mosaic:
  1. Acessar Multi-Screen Mosaic: No menu superior, clique em **"Control"** e selecione **"Multi-Screen Mosaic"**.
  2. Configurar Disposição:
    • Defina como as diferentes telas serão arranjadas (ex: 1x2, 2x2, etc.).
    • Configure a resolução total do mosaico e os parâmetros de cada tela individualmente.
  3. Observação Importante: As controladoras TB2 e TB20 PLUS não possuem saída HDMI e, portanto, não suporta a funcionalidade de mosaico de telas.
Obs: Não é possível com a TB2 e TB20 PLUS criar mosaico de telas, pois as controladoras não possui saída HDMI.

Pré-visualização

A pré-visualização é uma ferramenta crucial para verificar a aparência e o comportamento do conteúdo antes de publicá-lo no painel de LED. Ela garante que tudo esteja conforme o esperado.

Como utilizar a Pré-visualização:
  1. Localizar: A área de pré-visualização geralmente está disponível na tela de edição de layouts, próxima à opção "View Schedule".
  2. Simular Exibição: Ao clicar em "Preview", o software simulará como as imagens, vídeos, textos e transições serão exibidos no painel de LED.
  3. Ajustar: Utilize essa funcionalidade para fazer quaisquer ajustes de design, tempo ou animação antes de enviar o conteúdo para o display.

Tela Inicial - NovaLCT

A interface do NovaLCT foi projetada para facilitar o gerenciamento, configuração e manutenção de painéis LED de maneira eficiente. Este guia apresenta uma visão detalhada e acessível das funções básicas para que você possa utilizar o software de forma plena.

  1. Acesso Inicial:

    Ao abrir o NovaLCT, você será direcionado para a tela inicial. Caso necessário, atualize para a versão mais recente através da janela de atualização automática. Para se conectar a um painel LED:

    • No menu superior, clique em User > Media Player Login.
    • Na janela Terminal List, clique em Refresh para localizar dispositivos disponíveis.
    • Selecione o painel desejado e clique em Connect System. Insira as credenciais padrão (usuário: "admin", senha: "123456") ou personalizadas.
    • Se o painel não aparecer, clique em Specify IP, insira o endereço IP do painel manualmente e pressione Search.
  2. Menu Superior:

    O menu superior é o ponto central para acessar todas as funções principais:

    • System (S): Permite configurar o sistema, carregar arquivos de configuração e gerenciar dispositivos conectados.
    • Settings (C): Ajustes detalhados, incluindo resolução, frequência de atualização e parâmetros avançados das telas.
    • Tools (T): Acesse ferramentas de diagnóstico e manutenção, como ajustes de brilho e correção de pixels defeituosos.
    • Plug-in (P): Recursos adicionais para integrar novos módulos e funcionalidades extras.
    • User (U): Gerencie contas e permissões de usuários.
    • Language (L): Altere o idioma da interface conforme sua preferência.
    • Help (H): Acesso direto à documentação oficial, tutoriais e suporte técnico.
  3. Funções Principais:

    A tela inicial apresenta atalhos para as funções mais usadas. Aqui estão as principais:

    • Cloud Monitoring: Monitore o status dos painéis LED remotamente, verificando conectividade e recebendo alertas em tempo real.
    • Brightness: Ajuste manual ou automaticamente o brilho das telas para adequar às condições de luminosidade do ambiente.
    • Screen Control: Configure layouts personalizados, resoluções e parâmetros de posicionamento dos painéis LED.
    • Monitoring: Exibe informações essenciais de hardware, como temperatura, voltagem e status de conexão.
    • Multi-function Card: Gerencie cartões multifuncionais conectados aos painéis, otimizando suas configurações específicas.
    • Test Tool: Ferramenta dedicada para testes de desempenho, verificação de pixels defeituosos e sincronização das telas.
  4. Painel Central:

    Localizado no centro da interface, este painel exibe informações detalhadas sobre os dispositivos conectados:

    • Informações do Sistema Local: Número de controladores conectados, dispositivos adicionais e a versão do serviço.
    • Monitoramento:
      • Indicadores de status para cada dispositivo (verde para online e cinza para offline).
      • Visualização do hardware conectado, como controladores e sensores de temperatura.
    • Detalhes dos Dispositivos:
      • Porta de Comunicação: Mostra o IP e porta de comunicação do dispositivo conectado.
      • Tipo de Dispositivo: Identifica o modelo e especificações do controlador ou equipamento.
      • Número de Dispositivos: Quantifica dispositivos ativos na conexão.
      • S/N (Número de Série): Exibe o número de série do hardware.
      • Operações: Área destinada a comandos ou ações específicas para cada dispositivo.
    • Logout:

      Para encerrar sua sessão com segurança, clique no botão de logout localizado no canto superior direito. O nome do usuário logado também será exibido para fácil identificação.

Dica: Utilize a funcionalidade "Load Configuration File" para importar configurações previamente salvas e economizar tempo na configuração inicial. Salve alterações regularmente para evitar a perda de dados importantes.

Diagnóstico e Resolução de Problemas

Abaixo estão descrições detalhadas e soluções para problemas comuns encontrados no ViPlex e NovaLCT:

  • Conexão de Rede Instável:

    Descrição: O software não consegue detectar ou conectar-se ao painel LED. Isso pode ocorrer devido a problemas na rede ou configurações incorretas.

    Solução: Verifique os cabos Ethernet e certifique-se de que estão conectados corretamente. Configure endereços IP na mesma faixa e, se necessário, desative temporariamente o firewall para testar a conexão.

  • Falhas na Reprodução de Conteúdo:

    Descrição: O conteúdo de vídeo ou imagem apresenta falhas ou atrasos durante a reprodução.

    Solução: Verifique a compatibilidade dos formatos de mídia com o software. Atualize para a versão mais recente do ViPlex Express e revise as configurações de agendamento e reprodução.

  • Problemas de Calibração:

    Descrição: O brilho e as cores exibidos nos painéis não correspondem às configurações desejadas.

    Solução: Utilize as ferramentas de calibração do NovaLCT para ajustar o brilho e a cromaticidade. Inspecione os módulos LED para identificar possíveis falhas de hardware.

  • Erros de Sincronização:

    Descrição: Os horários de reprodução agendados não coincidem com os configurados no software.

    Solução: Sincronize os relógios do sistema e do painel LED. Verifique se o fuso horário está configurado corretamente em ambos os dispositivos.

Calibração de Brilho

  • Configurar brilho e contraste dos displays LED.

    Passo 5: Verifique o resultado final e ajuste conforme necessário para obter a melhor qualidade visual.

    Observação: Certifique-se de que o painel LED esteja conectado e operando corretamente antes de iniciar a calibração.

    Definir Horários de Exibição de Mídia

    O gerenciamento de layouts permite programar diferentes disposições de conteúdo para serem exibidas em horários específicos. Siga os passos abaixo para configurar:

    1. Abrir o Software:

      Inicie o Viplex e conecte-se ao painel LED utilizando as credenciais configuradas.

    2. Acessar Configuração de Layout:

      Na interface principal, vá até "Solutions" e selecione a opção "New > Regular Screen".

    3. Configurar Elementos do Layout:

      Adicione os seguintes elementos ao layout:

      • Imagens ou Vídeos: Arraste e solte os arquivos desejados no painel de edição.
      • Texto: Insira caixas de texto para adicionar mensagens dinâmicas ou fixas.
      • Relógios ou Temporizadores: Escolha widgets interativos para exibição de horários ou contagens regressivas.
    4. Definir Horários de Exibição:

      No menu "View Schedule", configure os horários de início e término para cada layout. Certifique-se de que não há sobreposição de horários.

    5. Pré-visualizar e Salvar:

      Clique na opção "Preview" para verificar como o layout será exibido no painel LED. Após confirmar, salve as configurações.

    6. Enviar para o Painel:

      Utilize a função "Publish" para aplicar o layout configurado ao painel.

    7. Beginner's Guide:

      Para uma introdução mais detalhada, consulte o guia de início rápido para configurar layouts ao lado esquerdo da função "View Schedule".

    Dica: Use layouts diferentes para horários de pico e horários de menor movimento, adaptando o conteúdo para o público-alvo.
