Bem-vindo ao Manual das Controladoras de LED VX

Este manual fornece informações detalhadas sobre a instalação, configuração e operação das controladoras de LED VX16s, VX400, VX600 e VX1000.

V-Can Video Control Software

O V-Can é um software profissional desenvolvido para o gerenciamento e controle detalhado de vídeos em displays LED. Compatível com controladoras da linha VX (como VX400, VX600 e VX1000), ele permite:

Instalação e Configuração

  1. Requisitos do Sistema:
    • Windows 10/11 (64-bit)
    • Processador Intel i5 ou superior
    • 8GB de RAM
    • Placa de vídeo com suporte a OpenGL 2.0
  2. Passo a Passo da Instalação:
    1. Baixe o instalador mais recente do site oficial
    2. Execute o arquivo de instalação
    3. Siga as instruções do assistente de instalação
    4. Reinicie o computador após a instalação
  3. Configuração Inicial:
    • Conecte a controladora ao computador via cabo de rede
    • Abra o software V-Can
    • Selecione o idioma preferido
    • Configure as preferências de rede
  • Ajuste fino das entradas de vídeo (resolução, taxa de quadros e configurações de cor);
  • Operação em modos distintos: Modo Direto para alterações imediatas e Modo Locação para pré-visualização antes de aplicar as configurações no painel;
  • Sincronização de múltiplos dispositivos, possibilitando a criação de layouts em Mosaic para grandes painéis;
  • Gerenciamento de fontes de vídeo com recurso de arrastar e soltar, facilitando a criação e edição de layouts personalizados;
  • Salvamento e carregamento de presets, otimizando a operação para diferentes cenários de instalação.

O V-Can é uma ferramenta indispensável para instalações que exigem controle preciso e flexível, garantindo a qualidade e a sincronização perfeita da exibição dos conteúdos.

Controladores LED

Conheça os modelos disponíveis e suas especificações técnicas:

Os controladores LED da linha VX são dispositivos de alta performance projetados para gerenciar displays LED de diversos tamanhos e complexidades. Abaixo estão as principais características e especificações técnicas:

Controlador VX16s

  • Aplicação: Ideal para pequenos displays LED
  • Resolução Máxima: 1920x1080 pixels
  • Taxa de Atualização: Até 60Hz
  • Interface: HDMI, DVI, VGA
  • Consumo: 15W

Controlador VX400

  • Aplicação: Médias telas LED
  • Resolução Máxima: 3840x2160 pixels
  • Taxa de Atualização: Até 120Hz
  • Interface: HDMI 2.0, DisplayPort
  • Consumo: 25W

Controlador VX600

  • Aplicação: Grandes painéis LED
  • Resolução Máxima: 4096x2160 pixels
  • Taxa de Atualização: Até 144Hz
  • Interface: HDMI 2.1, DisplayPort 1.4
  • Consumo: 35W

Controlador VX1000

  • Aplicação: Mega displays LED
  • Resolução Máxima: 7680x4320 pixels
  • Taxa de Atualização: Até 240Hz
  • Interface: HDMI 2.1, DisplayPort 1.4, SDI
  • Consumo: 50W

Instalação e Configuração

Dica: Antes de iniciar a instalação, verifique se todos os cabos estão corretamente conectados e se a controladora está ligada.

Conexões

Via Cabo de Rede

Esta configuração utiliza uma conexão física entre a controladora VX e o computador para garantir estabilidade e alta velocidade de transmissão. Siga os passos abaixo:

  1. Conecte o Cabo de Rede:

    Insira uma extremidade do cabo de rede na controladora VX e a outra no roteador ou diretamente no computador.

  2. Configuração de Rede:

    No computador, certifique-se de que a conexão está configurada para obter um endereço IP automaticamente (DHCP) ou configure manualmente, se necessário.

  3. Abra o Software:

    Inicie o V‑Can e verifique se a controladora foi detectada na lista de dispositivos.

  4. Teste a Conexão:

    Envie um conteúdo de teste para confirmar que a comunicação entre o computador e o painel LED está funcionando corretamente.

LAN com Fio

A configuração LAN com fio é ideal para redes locais e ambientes corporativos. Siga os passos abaixo:

  • Conecte o Cabo de Rede:

    Conecte uma extremidade do cabo na controladora VX e a outra no switch ou roteador da rede local.

  • Desative o DHCP (se necessário):

    Se preferir, desative o DHCP na controladora e configure um endereço IP estático.

  • Configuração de Rede no Computador:

    No computador, configure um endereço IP na mesma faixa da controladora VX.

  • Conexão com o Software:

    Abra o V‑Can e conecte-se à controladora utilizando o endereço IP configurado.

  • Verificação:

    Envie um conteúdo de teste para confirmar que a controladora está respondendo corretamente.

Wi-Fi AP

O modo Wi-Fi AP permite que a controladora VX funcione como um ponto de acesso, facilitando conexões sem fio. Siga os passos abaixo:

  • Localize o SSID:

    Identifique o SSID da controladora (geralmente "AP" seguido de um código único). Essa informação pode estar disponível na etiqueta ou no manual da controladora.

  • Conecte-se ao Wi-Fi:

    No computador ou dispositivo móvel, conecte-se à rede Wi-Fi usando o SSID e a senha padrão da controladora.

  • Abra o Software:

    Inicie o V‑Can e verifique se a controladora aparece na lista de dispositivos conectados via rede sem fio.

  • Configuração Inicial:

    Acesse as configurações da controladora e, se necessário, altere o SSID ou a senha para maior segurança.

  • Teste a Conexão:

    Envie um conteúdo de teste para confirmar que a transmissão está funcionando corretamente.

Conexão V‑Can Via Cabo de Rede

A conexão por cabo de rede é recomendada para desempenho máximo e estabilidade. Siga as etapas abaixo:

  • Conecte o Cabo: Insira uma extremidade do cabo na controladora VX e a outra no roteador ou computador.
  • Configuração de Rede: Configure o computador para obter um endereço IP automaticamente (DHCP) ou manualmente na mesma faixa da controladora.
  • Abra o Software: Inicie o V‑Can e verifique se a controladora aparece na lista de dispositivos.
  • Teste a Conexão: Envie um conteúdo de teste para confirmar o funcionamento correto da conexão.

Conexão V‑Can Via Wi-Fi AP

O modo Wi-Fi AP permite configurar a controladora VX como um ponto de acesso. Siga os passos abaixo:

  1. Ative o Wi-Fi AP: Verifique nas configurações da controladora se o modo AP está ativado e identifique o SSID.
  2. Conecte-se à Rede: No computador ou dispositivo móvel, conecte-se à rede Wi-Fi gerada pela controladora.
  3. Configuração Inicial: Acesse o V‑Can e ajuste as configurações de rede, como troca de senha ou personalização do SSID, se necessário.

Modelos Compatíveis

O V‑Can é compatível com os seguintes modelos de controladoras da linha VX:

  • VX400: Ideal para pequenos displays LED, com suporte a resolução padrão.
  • VX600: Suporte para telas médias e funcionalidades adicionais de ajuste de cores.
  • VX1000: Perfeito para grandes painéis, com capacidade de processamento para vídeos em alta definição.

Cada modelo oferece recursos específicos para atender a diferentes necessidades, garantindo flexibilidade e alto desempenho.

Operação e Funcionalidades

Operação

Tela Inicial - V-Can

A interface do V-Can foi projetada para facilitar a operação e o gerenciamento dos displays LED. Aqui estão as principais áreas e funcionalidades:

  1. Barra Superior:
    • File: Criar, abrir, salvar e exportar arquivos de configuração.
    • Settings: Ajustes avançados, como idioma, sincronização de tempo, parâmetros de rede e modos de exibição.
    • Tools: Ferramentas adicionais para diagnóstico, testes e calibração.
    • Help: Acesso à documentação oficial, suporte técnico e verificação de atualizações do software.
  2. Painel Central:

    Exibe informações detalhadas sobre os dispositivos conectados, como:

    • Status do dispositivo: Informações sobre conexão, IP e versão do firmware.
    • Gerenciamento de fontes de vídeo: Lista de entradas disponíveis (HDMI, DP, DVI, etc.) e ajustes em tempo real.
    • Configurações de resolução: Ajuste de tamanho, escala e proporção da imagem exibida.
    • Alertas e logs: Diagnóstico de possíveis falhas e mensagens de status do sistema.
  3. Menu Lateral: Acesso rápido a funções essenciais para otimizar a operação:
    • Camadas e Layouts: Gerenciamento de múltiplas camadas de vídeo e presets de configuração.
    • Calibração de Cores: Ajustes finos de brilho, contraste, temperatura de cor e saturação.
    • Sincronização e Modo Mosaic: Controle de múltiplos dispositivos para composições em telas grandes.
    • Configuração de Saída: Definição de parâmetros de saída para compatibilidade com diferentes painéis LED.

    Guia Passo a Passo para Uso Avançado

    Gerenciamento de Fontes de Vídeo

    1. Na barra superior, clique em "Sources"
    2. Selecione o tipo de fonte desejada (HDMI, DVI, etc.)
    3. Arraste a fonte para a área de exibição desejada
    4. Ajuste as propriedades da fonte nas configurações laterais

    Criação de Layouts Personalizados

    1. No menu lateral, selecione "Layouts"
    2. Escolha entre layouts predefinidos ou crie um novo
    3. Arraste e solte as fontes nas áreas desejadas
    4. Ajuste o tamanho e posição de cada área
    5. Salve o layout como um preset para uso futuro

    Calibração de Cores

    1. No menu lateral, selecione "Color Calibration"
    2. Escolha o modo de calibração (Manual ou Automático)
    3. Para calibração manual, ajuste os valores de:
      • Brilho
      • Contraste
      • Temperatura de cor
      • Saturação
    4. Salve o perfil de cores para uso futuro

    Sincronização de Múltiplos Dispositivos

    1. Conecte todos os dispositivos na mesma rede
    2. No menu "Devices", selecione todos os dispositivos
    3. Escolha o modo de sincronização (Mosaic ou Espelhamento)
    4. Configure as áreas de exibição para cada dispositivo
    5. Teste a sincronização com conteúdo de teste

    Gerenciamento de Presets

    1. Configure todas as preferências desejadas
    2. No menu superior, selecione "File" > "Save Preset"
    3. Nomeie o preset de forma descritiva
    4. Para carregar, selecione "File" > "Load Preset"
    5. Organize os presets em pastas para fácil acesso
